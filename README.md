하지만 입력이 큰 하나의 테스트 케이스에 대해서

이 문제의 경우 테스트 입력을 일일이 손으로 입력하기 번거로워서
처음으로 Scanner 대신 파일 입출력을 이용하였는데
매우 편리했다. 종종 써먹어야겠다.
